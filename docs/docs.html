Getting Started

foot.css is available via CDN you just need to add this tag in your project head.

<link rel="stylesheet" href="https://unpkg.com/footcss">

foot.css is also available using npm to install it directly into your project.

$ npm i footcss

Components

Simple Grid System

foot.css provides you a fluid grid system that resize with the viewport. Wrap your content in a .container to center it on the page.

.foot-col
.foot-col
.foot-col
.foot-col

    <div class="foot-row">
        <div class="foot-col"> .foot-col </div>
        <div class="foot-col"> .foot-col </div>
        <div class="foot-col"> .foot-col </div>
    </div>
    <div class="foot-row">
        <div class="foot-col"> .foot-col </div>
    </div>
Buttons

foot.css includes predefined button styles, with a distinctive look. The buttons support icons as well with the help of the Font Awesome library, that you will need to add to your document head.

Button
